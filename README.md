# 📸 Snapshot - React + Tailwind CSS Image Search

**Snapshot** is a simple and elegant image search app built with **React** and **Tailwind CSS v4**.  
It lets you search for beautiful images using the **Unsplash API**.  
Type your own search keywords or use quick searches to find inspiring photos instantly!

## 🚀 Features

- 🔍 **Search Images** — Enter any keyword to fetch high-quality images from Unsplash.
- ⚡ **Quick Searches** — Try popular searches with one click.
- 🎨 **Modern UI** — Styled with Tailwind CSS v4 for a responsive, clean design.
- 🌐 **API Integration** — Uses the Unsplash API to fetch real-time image data.
- ✅ **Built with React** — Uses reusable components and state management.
